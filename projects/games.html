Projects / Info

During the Winter I took an experimental Game Art course with Rick Silva . This was my first exposure to using game engines and 3D programs. I plan on experimenting with them much more in the future.

The first game I made is called Enigma Stones. It is playable on itch.io. Unfortunately, it is demanding on the CPU and GPU and runs very slowly in the browser.

My second game, Infinite Raver, is still in beta. A complete open version of the game is available on github.com. The game is playable but there are efficency issues when you keep the game open for long periods of time. If you are interested in collaborating please send me an email.
